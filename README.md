# 📚 Projetos de Desenvolvimento Mobile - UEG

Este repositório contém os projetos desenvolvidos para a disciplina de Desenvolvimento Mobile

---

## 🚀 Projetos Integrados

### 1. Calculadora Estilo Samsung S24
Calculadora aritmética que replica a estética do Samsung S24.
- **Tecnologias:** React Native, JavaScript.

### 2. Tradutor 
Aplicação que consome a API externa `MyMemory` para realizar tradução de textos de Português para Inglês.
- **Operação:** Processamento de string via requisição assíncrona (Axios).
- **Destaques:** Interface Dark Mode com detalhes em roxo, tratamento de estados de carregamento (loading) e integração com API REST.
- **Tecnologias:** React Native, Axios, API MyMemory.

## 🛠️ Como executar os projetos
1. **Pré-requisitos:**
   - Node.js instalado (v20+ recomendado via NVM)
   - Aplicativo **Expo Go** no celular ou Emulador Android

2. **Instalação:**
   ```bash
   # Clone o repositório
   git clone [https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git](https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git)

   # Entre na pasta do projeto desejado
   cd projeto-tradutor  # ou cd projeto-calculadora

   # Instale as dependências
   npm install

3. **Rodar o App:**
   ```bash  
   npx expo start
