# 🌐 Fynx Tradutor - Modo Dark

Uma aplicação mobile desenvolvida com React Native que realiza a tradução de textos do Português para o Inglês em tempo real, utilizando integração com a API MyMemory.

## 🚀 Funcionalidades
- **Tradução Inteligente:** Consome uma API REST para processamento linguístico.
- **Interface Moderna:** Estética baseada na One UI da Samsung, com modo escuro e detalhes em roxo.
- **Arquitetura Modular:** Separação clara entre estilos, componentes e lógica de serviço.

## 🛠️ Tecnologias
- **React Native / Expo**
- **Axios** (Cliente HTTP)
- **MyMemory API** (Serviço de Tradução)

## 📁 Estrutura do Projeto
- `src/components`: Componentes visuais reutilizáveis (Botão, Card).
- `src/services`: Lógica de integração com a API.
- `src/styles`: Centralização da identidade visual e temas.
